SEdictionary API

This is the documentation of our SEdictionary API. Given a software-specific term, it can return the definition and related words for the term.


Usage of the API

This API returns data in json format.

As one abbreviation may represent several full names, so there may be several items in the return file.

For each software-specific term, there are several attributes:

  • the normalization name for the term;
  • its Wikipedia link (optional);
  • top relevant words and corresponding Wikipedia links (optinal).

Be aware that not all terms have Wikipedia link or relevant words.

For exmaple, you can get the dictionary of "javascript" by querying /api/javascript.
